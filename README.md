# Flight Ticket Booking System

A simple **flight ticket booking web application** built with **Go (Golang)** and HTML/CSS.  
This demo app allows users to **search flights, book tickets, and see booking confirmation**, with a clean and modern UI.

---

## Features

- Search for flights by **departure and arrival city**  
- View **available flights** with airline, flight number, departure/arrival times, and price  
- Book a flight by entering **passenger information**  
- View **booking confirmation** with **total price calculation**  
- Responsive, professional, and modern design using HTML/CSS  
- Go templates with **custom function** for total price calculation
